## Keys

This tool based on [Cohere](https://cohere.com/) LLM service. To enable CLI working you should expose [api key](https://dashboard.cohere.com/api-keys) as env variable and enjoy.
```
export YASHI_COHERE_KEY='gYjIU2+....'
```
